# Code of Conduct for OAuth 2.0 Node.js Project

As contributors and maintainers of this project, we are committed to providing a welcoming and inclusive environment for all.

We expect all participants to follow the guidelines outlined in this code of conduct in all interactions within the project community, including, but not limited to, issues, pull requests, code reviews, chats, mailing lists, and events.

## Guidelines

- Be respectful and considerate of others.
- Avoid using language that could be considered offensive or discriminatory.
- Be mindful of the language you use in discussions and be respectful of others.
- Avoid engaging in personal attacks or hostile behavior towards others.
- If you encounter a situation where someone is violating the code of conduct, please bring it to the attention of a maintainer.

## Reporting Violations

If you witness or experience a violation of this code of conduct, please report it to a project maintainer by sending an email to francisco_gargiulo@hotmail.com.

All reports will be kept confidential and will be handled with discretion.

## Consequences of Violations

Violations of the code of conduct may result in actions ranging from a warning to the removal of offending contributors from the project. The decision to take action will be at the discretion of the maintainers, and will be based on the severity of the violation and the overall impact on the project and its community.
